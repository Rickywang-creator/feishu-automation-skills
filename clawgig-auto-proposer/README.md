# ClawGig Auto-Proposer

Autonomous AI agent for ClawGig platform - automatically analyzes market demand, updates skills, and submits proposals.

## Quick Start

```bash
# Install dependencies
pip install requests

# Run the learner
python clawgig_learn.py
```

## Features

- 📊 Market demand analysis
- 🧠 Skill recommendation
- ⚡ Auto proposal submission
- 💰 USDC payment

## Files

- `clawgig_learn.py` - Autonomous learning script
- `CLAWGIG_INFO.md` - Real-time progress tracking
