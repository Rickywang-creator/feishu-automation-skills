# Data Cleaning Specialist

> 数据清洗专家，解决数据杂乱、格式不统一、重复记录等痛点。

## 功能特性

### 核心能力
1. **去重** - 自动检测并合并重复记录
2. **格式化** - 统一日期、数字、文本格式
3. **缺失值处理** - 自动填充默认值或标记
4. **数据验证** - 检查数据有效性（邮箱、手机号、身份证等）
5. **批量清洗** - 支持 10000+ 条记录的批量处理
6. **格式转换** - CSV/Excel/JSON/JSONL 互转

### 适用场景
- Excel 数据清洗和整理
- CSV/JSON 数据规范化
- 历史数据格式统一
- 多表格数据合并
- 数据质量检查和修复

## 使用方法

### 基本清洗
```
帮我清洗这个数据：
- 文件：data.csv
- 清洗规则：去重 + 格式统一
- 输出：cleaned.csv
```

### 指定规则
```
按以下规则清洗：
1. 电话号码统一格式（138xxx）
2. 日期转为 YYYY-MM-DD
3. 删除空白行
4. 邮箱格式验证
5. 去除前后空格
```

### 批量处理
```
批量处理：
- 文件夹：./data/
- 100个CSV文件合并成一个
- 按日期重命名
- 生成汇总报表
```
