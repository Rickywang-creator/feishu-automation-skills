# feishu-bitable-sync-alert - 飞书多维表格定时提醒

> 定时检查多维表格数据，自动推送提醒通知。

## 功能特性

### 核心能力
1. **定时检查** - 每天/每周定时检查数据
2. **变更提醒** - 数据新增/修改时立即通知
3. **阈值告警** - 超过设定值时提醒
4. **待办提醒** - 定时提醒待处理事项
5. **汇总推送** - 每日/每周数据汇总

### 适用场景
- 销售数据每日汇总
- 待办事项提醒
- 库存预警
- 客户跟进提醒
- KPI 达标通知

## 使用方法

### 设置提醒
```
设置每天上午9点提醒：
- 检查表格：销售数据
- 检查内容：昨日新增客户数
- 提醒方式：飞书消息
```

### 数据变更通知
```
监控"客户跟进表"：
- 字段：下次跟进日期
- 条件：到期前1天提醒
- 推送：飞书通知
```

## 技术参数
- 检查频率：每小时/每天/每周
- 支持监控维度：5 个
- 响应时间：< 1 分钟

## 使用示例

### 示例 1：每日销售汇总
**设置**：每天 18:00 检查当日销售额
**结果**：自动推送销售汇总到飞书

### 示例 2：客户跟进提醒
**设置**：提前1天提醒"下次跟进日期"
**结果**：自动推送跟进提醒

### 示例 3：库存预警
**设置**：库存 < 100 时提醒
**结果**：库存不足立即通知

### 示例 4：KPI 告警
**设置**：销售额 < 目标 80% 时告警
**结果**：未达标时自动提醒

## 注意事项
1. 合理设置检查频率，避免打扰
2. 重要提醒需确认收到
3. 深夜勿扰（23:00-08:00 静音）
4. 建议先测试再正式使用
