# feishu-bitable-analyzer - 飞书多维表格数据分析

> 自动分析多维表格数据，生成统计报表和可视化。

## 功能特性

### 核心能力
1. **自动统计** - 求和/平均/计数/最大值/最小值
2. **数据透视** - 多维度交叉分析
3. **趋势分析** - 时间序列数据趋势
4. **异常检测** - 自动标记异常数据
5. **报表生成** - 自动生成数据分析报告

### 适用场景
- 销售数据统计和分析
- 业务数据月度/季度汇总
- 客户数据分析
- 绩效数据统计

## 使用方法

### 自动分析
```
分析这个多维表格：
- 表格：xxx
- 分析维度：按部门统计销售额
- 输出：统计表 + 趋势图
```

### 生成报表
```
生成月度销售报表：
- 包含：各地区销售额、毛利、客户数
- 格式：数据透视表 + 趋势图
```

## 技术参数
- 分析维度：最多 5 个
- 支持数据量：1-10000 条
- 处理速度：约 1 秒/100 条

## 使用示例

### 示例 1：销售统计
**输入**：销售记录表
**输出**：各产品销售额、毛利、占比

### 示例 2：客户分析
**输入**：客户信息表
**输出**：客户分布、活跃度、留存率

### 示例 3：趋势报告
**输入**：3 个月销售数据
**输出**：月度趋势、同比增长、预测

2. 复杂分析需要明确需求
3. 结果可人工复核
