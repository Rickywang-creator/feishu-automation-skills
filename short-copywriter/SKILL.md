## 核心功能

### 1. 朋友圈文案生成
**输入**：产品/服务、目标人群、促销力度
**输出**：3-5条不同风格的朋友圈文案
**风格选项**：
- 真实分享：第一人称，亲身经历
- 硬广推广：产品卖点，促单导向
- 软种草：场景代入，不提产品
- 问答互动：提问引导评论
- 活动造势：限时限量紧迫感

### 2. 小红书种草笔记
**输入**：产品、亮点、目标人群
**输出**：完整种草笔记
**结构**：
1. 痛点场景开头（3秒抓住注意力）
2.解决方案展示（真实使用感）
3. 细节外化（不用情绪词，用画面）
4. 行动引导（评论/收藏/关注）

**万能开头公式**：
- 「[人群]必看！这个[产品]让我[结果]」
- 「救命！终于找到[痛点]的克星了」
- 「别再XXX了，试试这个方法」

### 3. 产品卖点翻译
**输入**：产品卖点
**输出**：用户能听懂的好处
**翻译公式**：
```
❌ 卖点：玻尿酸含量99%
✅ 好处：敷15分钟，像给脸灌了1000ml冰美式，毛孔直接原地隐身
```
**核心技巧**：把卖点翻译成画面，用户脑子里能想象出来

### 4. 活动宣传文案
**输入**：活动类型、优惠力度、截止时间
**输出**：活动文案
**紧迫感营造**：
- 限时：仅剩X个名额
- 限量：限XX件
- 折扣：X折/买X送X

### 5. 评论互动回复
**输入**：用户评论内容
**输出**：高情商回复
**原则**：
- 顺着说，不硬杠
- 给台阶，不戳穿
- 留钩子，引导下单

## 工作流程

### 标准流程（4步）
1. **明确需求**：文案类型 + 目标 + 产品
2. **选择风格**：参考上面的风格选项
3. **生成内容**：AI 生成 3-5 条备选
4. **选择微调**：pick最合适的，简单调整发

### 快速流程（2步）
1. **一句话需求**：「帮我发个卖XX的朋友圈」
2. **获取结果**：直接拿文案发

## 文案急救箱

### 万能情绪痛点词
累、烦、慌、怕、亏、难、僵、焦虑、迷茫、内耗、失望、无助、不甘心

### 高转化句式
1. 「不是XXX，而是XXX」
2. 「 XX 都XXX了，你还在XXX」
3. 「XX的尽头是XXX」
4. 「自从XXX后，XX都XXX了」
5. 「救命！XXX太XXX了」

### 留钩子技巧
- 结尾问问题：「你们觉得哪个好看？评论告诉我」
- 引导收藏：「收藏这篇，下次找不到」
- 行动号召：「要的扣1，我来私你」
