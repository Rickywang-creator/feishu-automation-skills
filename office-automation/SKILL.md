# Office Automation Specialist

> 办公自动化专家，一站式解决 Excel/PPT/Word/文档处理痛点。

## 功能特性

### 核心能力
1. **Excel 自动化** - 公式、宏、数据处理、图表生成
2. **PPT 制作** - 自动生成、模板套用、批量处理
3. **Word 文档** - 批量生成、格式统一、内容提取
4. **飞书文档** - 自动创建、批量处理、内容摘要
5. **数据汇总** - 多文件合并、跨表汇总、报表生成

### 适用场景
- 每周/月报自动生成
- 批量处理Excel数据
- PPT模板批量套用
- 会议纪要自动整理
- 数据跨平台迁移

## 使用方法

### Excel 任务
```
帮我处理Excel：
- 文件：sales.xlsx
- 任务：按月份汇总、生成图表
- 输出：报表.xlsx
```

### PPT 任务
```
帮我制作PPT：
- 主题：年度总结
- 页数：10页
- 风格：简约商务
```

### 文档任务
```
批量处理文档：
- 文件夹：./docs/
- 任务：统一格式、添加页眉
- 输出：./output/
```

## 技术参数
- Excel：支持公式、VBA、图表、数据透视表
- PPT：支持模板、母版、动画
- Word：支持样式、目录、页眉页脚
- 批量：支持 100+ 文件同时处理
