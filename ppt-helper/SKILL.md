## 核心功能

### 1. 大纲生成器
**输入**：主题、目标受众、演示目的
**输出**：结构化大纲（10-15页）
**工作流**：
1. 理解主题核心
2. 确定演示逻辑（总-分-总/问题-方案-结果/故事-高潮-行动）
3. 生成页面规划
4. 每页要点提炼

### 2. 模板推荐引擎
**输入**：演示场景、行业、风格偏好
**输出**：推荐模板 + 适配建议
**分类**：
- 商业融资：专业、简洁、数据可视化强
- 工作汇报：清晰、层级分明、图表为主
- 产品发布：创意、视觉冲击、故事线
- 培训课件：结构化、要点突出、案例丰富

### 3. 配色方案库
**输入**：行业、场景、情感基调
**输出**：配色方案（主色+辅色+强调色）
**内置方案**：
- 科技蓝：#0066CC + #003366 + #00CCFF
- 商务黑：#333333 + #666666 + #FF6600
- 温暖橙：#FF6600 + #FFCC00 + #333333
- 清新绿：#00AA55 + #88DD88 + #333333

### 4. 快速排版工具
**内容替换**：输入文字，自动排版
**批量调整**：字体/配色/版式一键统一
**图片适配**：自动裁剪、调整比例

### 5. 批量修改指令
**改字体**：指定页面范围，一次性替换
**改配色**：全局替换配色方案
**改版式**：指定模板，全套应用

## 工作流程

### 标准流程（5步）
1. **明确需求**：主题、目的、页数、受众
2. **生成大纲**：AI 生成结构化页面规划
3. **选择模板**：根据场景推荐最佳模板
4. **填充内容**：每页要点 + 配图建议
5. **批量调整**：统一排版、配色、字体

### 快速流程（3步）
1. **输入主题**：一句话说明做什么演示
2. **选择场景**：融资/汇报/发布/培训
3. **获取结果**：完整大纲 + 模板推荐

## 适用人群

- 频繁做 PPT 的人（每周3次+）
- 创业者（融资路演、商业计划书）
- 管理者（工作汇报、季度总结）
- 销售/运营（客户演示、方案展示）
- 任何人需要「紧急出 PPT」的场景


## 边界与限制

- **不做的**：直接生成 PPT 文件（可生成大纲文档）
- **不教的基础**：如何新建 PPT、如何插入图片（找办公软件助手）
- **版权注意**：使用原创模板，参考但不照抄商业模板
