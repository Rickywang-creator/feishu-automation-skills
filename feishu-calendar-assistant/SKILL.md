# feishu-calendar-assistant - 飞书日程管理助手

> 智能管理飞书日程，自动提醒和安排。

## 功能特性

### 核心能力
1. **日程创建** - 自然语言创建日程
2. **时间查找** - 智能查找空闲时间
3. **冲突检测** - 检测时间冲突
4. **自动提醒** - 多时段提醒
5. **日程汇总** - 每日/每周日程汇总

### 适用场景
- 日程自动创建
- 会议安排
- 提醒设置
- 时间协调

## 使用方法

### 创建日程
```
帮我安排：
- 下周一上午10点项目评审会
- 参与人：产品、技术、设计
- 提醒：提前15分钟
```

### 查找空闲
```
帮我找明天14:00-18:00的空闲时间：
- 参与人：张三、李四
```

### 设置提醒
```
今天的会议都设置提醒：
- 提前30分钟
- 方式：飞书消息
```

## 技术参数
- 支持日历数：10+
- 并发检测：10人
- 提醒时段：5/15/30/60分钟

## 使用示例

### 示例 1：创建会议
**输入**："周三下午3点技术评审会"
**输出**：
- 创建日程（周三15:00）
- 发送邀请

### 示例 2：冲突检测
**输入**："安排周五下午"
**输出**：检测到冲突，推荐其他时间

### 示例 3：智能提醒
**输入**："半小时后有会议"
**输出**：提前30分钟自动提醒

## 注意事项
1. 需要日历权限
2. 最好提前24小时安排
3. 定期清理过期日程
4. 重要日程多重提醒
