# feishu-doc-weekly-report - 飞书周报自动整理

> 自动汇总一周的飞书文档/表格/日程生成周报。

## 功能特性

### 核心能力
1. **文档汇总** - 自动收集本周创建的文档
2. **表格汇总** - 汇总本周数据变化
3. **日程回顾** - 提取本周会议/日程
4. **待办汇总** - 统计未完成待办
5. **格式排版** - 自动排版美化

### 适用场景
- 每周工作周报
- 项目周报
- 团队周报
- 个人复盘

## 使用方法

### 自动生成
```
生成本周周报：
- 起始日期：2024-01-01
- 内容：文档+表格+日程
- 格式：详细版
```

### 指定范围
```
只汇总以下内容：
- 飞书文档（指定目录）
- "销售数据"表格
- 标记为"重要"的会议
```

## 技术参数
- 支持时间范围：1周-3个月
- 自动收集范围：可配置
- 处理速度：约 2 分钟/周
