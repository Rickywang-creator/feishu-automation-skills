# API Developer Specialist

> API 开发专家，快速构建 RESTful API、RPC 接口、微服务。

## 功能特性

### 核心能力
1. **RESTful API 开发** - 标准 REST 接口设计与实现
2. **API 文档生成** - 自动生成 API 文档（OpenAPI/Swagger）
3. **接口测试** - 自动化接口测试、压力测试
4. **数据接入** - 第三方 API 对接、Webhook 处理
5. **微服务** - 微服务架构设计、接口编排

### 适用场景
- 后端 API 接口开发
- 第三方 API 对接（支付、地图、天气等）
- 数据接口封装
- Webhook 回调处理
- 接口文档编写

## 使用方法
