# feishu-bitable-cleaner - 飞书多维表格数据清洗专家

> 飞书多维表格数据清洗自动化，解决数据杂乱、格式不统一、重复记录等痛点。

## 功能特性

### 核心能力
1. **去重** - 自动检测并合并重复记录
2. **格式化** - 统一日期、数字、文本格式
3. **缺失值处理** - 自动填充默认值或标记
4. **数据验证** - 检查数据有效性（邮箱、手机号等）
5. **批量清洗** - 支持 1000+ 条记录的批量处理

### 适用场景
- Excel 导入飞书前的数据清洗
- 历史数据整理和格式化
- 多表格数据合并去重
- 数据质量检查和修复

## 使用方法

### 基本清洗
```
帮我清洗这个多维表格：
- 表格 token: xxx
- 表 ID: xxx
- 清洗规则: 去重 + 格式统一
```

### 指定规则
```
按以下规则清洗：
1. 电话号码统一格式（138xxx）
2. 日期转为时间戳
3. 删除空白行
4. 邮箱格式验证
```
