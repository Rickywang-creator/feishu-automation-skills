# feishu-doc-batch-format - 飞书文档批量处理

> 批量处理飞书文档，格式统一、批量排版。

## 功能特性

### 核心能力
1. **格式统一** - 批量统一文档格式
2. **批量排版** - 标题、段落、列表统一
3. **模板应用** - 应用文档模板
4. **批量替换** - 关键词批量替换
5. **目录生成** - 自动生成目录

### 适用场景
- 文档规范化
- 批量排版
- 模板应用
- 内容统一修改

## 使用方法

### 统一格式
```
统一这批文档的格式：
- 文档列表：xxx
- 格式：标题H1，正文14px
```

### 应用模板
```
把这10篇文档都改成模板格式：
- 模板：xxx
```

### 批量替换
```
把"AI"全部替换为"人工智能"：
- 范围：指定文档
```

## 技术参数
- 支持批量：1-100 篇
- 处理速度：约 10 秒/篇
- 格式选项：20+
