# Python Automation Scripts Specialist

> 自动化脚本专家，解决重复性工作、批量处理、定时任务等痛点。

## 功能特性

### 核心能力
1. **文件批量处理** - 批量重命名、移动、压缩
2. **数据定时抓取** - 定时爬取网页/API数据
3. **邮件自动处理** - 自动收发、分类、回复
4. **Excel/CSV 自动化** - 批量处理、数据转换
5. **定时任务** - 定时执行、状态监控

### 适用场景
- 每天重复的数据报表生成
- 批量文件整理和归档
- 定时抓取竞品价格/数据
- 自动备份和同步
- 定时发送消息/通知

## 使用方法

### 基本任务
```
帮我写一个Python脚本：
- 每天早上9点自动抓取XX网站数据
- 保存到Excel并发送邮件
```

### 批量处理
```
批量处理这个文件夹：
- 100个CSV文件合并成一个
- 按日期重命名
- 生成汇总报表
```

### 自动化流程
```
创建一个自动化流程：
1. 每小时检查XX API
2. 如果有新数据则下载
3. 清洗后存入数据库
4. 发送企业微信通知
```
