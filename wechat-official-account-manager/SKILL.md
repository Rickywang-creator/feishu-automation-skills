# 公众号运营官

> 不会写文章？不知道如何涨粉？输入主题，还你一篇专业公众号文章

## 核心能力

### 1. 文章结构工厂
- 开头：3 种模板（故事引入 / 数据冲击 / 观点鲜明）
- 框架：万能分段结构（是什么 → 为什么 → 怎么做）
- 结尾：行动引导 + 互动话题 + 往期推荐

### 2. 风格调节器
- 专业正式：行业分析 / 报告解读 / 深度观察
- 轻松亲切：生活分享 / 经验感悟 / 职场故事
- 幽默风趣：段子手 / 吐槽向 / 创意文案

### 3. 标题生成器
- 数字吸睛型（5 个技巧 / 3 个方法）
- 痛点共鸣型（你还在...吗？）
- 对比反差型（月薪 3 千 vs 月薪 3 万）
- 悬念引导型（原来这就是...）

### 4. 涨粉策略包
- 选题库：热点追踪 + 节日规划 + 行业日历
- 排版规范：字号 / 配色 / 配图 / 留白
- 互动设计：投票 / 问答 / 抽奖
- 推广渠道：朋友圈 / 社群 / 视频号导流

### 5. 数据分析
- 阅读量分析：标题 / 封面 / 选题
- 粉丝画像：年龄 / 地域 / 兴趣
- 转化分析：文章带货效果
- 竞品监控：对标账号分析

## 使用示例

```
用户输入：写一篇关于「年轻人躺平」的文章，目标是职场年轻人

输出：
【标题】
第一批 90 后已经开始躺平了

【正文结构】
开头：现象描述 + 数据支撑（某调查报告）
中间：
  - 躺平的定义（不只是不工作）
  - 为什么会躺平（房价/内卷/看不到希望）
  - 躺平的代价（没有保障/未来风险）
结尾：我的观点（可以卷，但要为自己卷）
互动话题：你觉得躺平是对的吗？

【排版建议】
标题：18px 加粗
正文：14-15px，1.8 倍行距
配图：与内容相关的高清图
```
