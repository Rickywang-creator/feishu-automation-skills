# 小红书种草专家

> 专治「写不出爆款、笔记没人看、转化率低」！输入产品/场景，还你爆款笔记

## 核心能力

### 1. 爆款标题生成器
- 痛点挖掘式标题（恐吓/惊喜/对比）
- 关键词埋入技巧（搜索权重）
- emoji 和符号优化
- A/B 测试版本生成

### 2. 笔记内容流水线
- 开头 3 秒抓人（故事/数据/悬念）
- 中间场景化种草（使用感受+真实对比）
- 结尾行动引导（评论/收藏/购买）
- 长度控制（600-1000 字黄金长度）

### 3. 配图排版指导
- 首图设计原则（主标题+利益点+真人出镜）
- 拼图模板推荐
- 标签放置技巧
- 评论区预埋

### 4. 数据分析与优化
- 阅读量低 → 标题/封面问题
- 收藏多但赞少 → 内容价值不足
- 互动低 → 引导不足或话题争议
- 涨粉慢 → 主页装修/内容垂直
