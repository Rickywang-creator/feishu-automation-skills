# feishu-bitable-auto-import - 飞书多维表格批量录入

> 自动化批量录入多维表格，支持多种数据源和定时同步。

## 功能特性

### 核心能力
1. **批量新增** - 支持 Excel/CSV 批量导入
2. **增量更新** - 只更新变化的数据
3. **定时同步** - 每天/每周定时执行
4. **多表同步** - 跨表格数据同步
5. **格式转换** - 自动转换数据格式

### 适用场景
- 每日销售数据自动录入
- 客户信息批量同步
- 跨平台数据迁移
- 定时报表生成

## 使用方法

### 批量导入
```
帮我批量导入数据到多维表格：
- 源文件：xxx（Excel/CSV/飞书文档）
- 目标表格：xxx
- 映射规则：列A→姓名、列B→电话
```

### 定时同步
```
设置每天上午9点自动同步：
- 源：飞书文档"销售日报"
- 目标：多维表格"销售数据"
- 同步方式：增量更新
```

## 技术参数
- 支持记录数：1-5000 条/批次
- 批量间隔：建议 50 条/批
- 并发数：6-8（最优）

## 使用示例

### 示例 1：Excel 导入
**输入**：Excel 文件含 100 条客户数据
**输出**：自动录入飞书多维表格

### 示例 2：增量同步
**输入**：每日更新的销售数据
**输出**：自动同步新增记录，保留历史

### 示例 3：定时任务
**设置**：每天 09:00 自动同步昨日数据
**结果**：无需手动，数据自动更新

## 注意事项
1. 建议非高峰期执行（避开 9:00-10:00）
2. 大文件分批导入
3. 关键操作保留人工确认环节
4. 定期检查同步状态
